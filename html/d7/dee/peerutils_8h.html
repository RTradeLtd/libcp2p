peer_id_t* libp2p_new_peer_id ( unsigned char *  output,
size_t *  output_len,
unsigned char *  input_hash,
size_t  input_size 
)

constructs a peer identifier from the given public key

this is responsible for taking the hash of a public key

turning it into a multihash, and the base encoding it

Note
is currently hard coded for the input hash being SHA256
if you allocated memory for output param you can free it up after this function returns
this is because we copy the data placed there into the returned struct
Parameters
outputwhere to store the resulting peerID
output_lenthe max lenght of the output buffer, but will also be used to store size of buffer
input_hashthe input public key hash, usually sha256
input_sizethe input size, usually 32 bytes
Returns
Success: pointer to an instance of peer_id_t
Failure: NULL pointer
