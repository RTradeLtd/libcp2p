libcp2p
A libp2p-like protocol with first class support for embedded systems. End goal is a protocol that can run on anything from your fridge, embedded systems, and anything else. Higher level languages bindings will be done via FFI. Heavy WIP

unsigned char* libp2p_crypto_ecdsa_keypair_peerid ( ecdsa_private_key_t *  pk)

returns the peerid for the corresponding private key

Warning
if hash of public key contains a 0, the output will be incorrect
caller must free returned pointer when no longer needed

to get the peerid we take a sha256 hash of the public key file in PEM format

we then generate a multihash of that sha256, and base58 encode it
