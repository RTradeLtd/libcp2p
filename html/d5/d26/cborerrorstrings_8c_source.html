31 /**
32  * \enum CborError
33  * \ingroup CborGlobals
34  * The CborError enum contains the possible error values used by the CBOR encoder and
35  * decoder.
36  *
37  * TinyCBOR functions report success by returning CborNoError, or one error
38  * condition by returning one of the values below. One exception is the
39  * out-of-memory condition (CborErrorOutOfMemory), which the functions for \ref
40  * CborEncoding may report in bit-wise OR with other conditions.
41  *
42  * This technique allows code to determine whether the only error condition was
43  * a lack of buffer space, which may not be a fatal condition if the buffer can
44  * be resized. Additionally, the functions for \ref CborEncoding may continue
45  * to be used even after CborErrorOutOfMemory is returned, and instead they
46  * will simply calculate the extra space needed.
47  *
48  * \value CborNoError No error occurred
49  * \omitvalue CborUnknownError
50  * \value CborErrorUnknownLength Request for the length of an array, map or
51  * string whose length is not provided in the CBOR stream \value
52  * CborErrorAdvancePastEOF Not enough data in the stream to decode item
53  * (decoding would advance past end of stream) \value CborErrorIO An I/O error
54  * occurred, probably due to an out-of-memory situation \value CborErrorGarbageAtEnd
55  * Bytes exist past the end of the CBOR stream \value CborErrorUnexpectedEOF End of
56  * stream reached unexpectedly \value CborErrorUnexpectedBreak A CBOR break byte
57  * was found where not expected \value CborErrorUnknownType An unknown type
58  * (future extension to CBOR) was found in the stream \value CborErrorIllegalType An
59  * invalid type was found while parsing a chunked CBOR string \value
60  * CborErrorIllegalNumber An illegal initial byte (encoding unspecified
61  * additional information) was found \value CborErrorIllegalSimpleType An illegal
62  * encoding of a CBOR Simple Type of value less than 32 was found \omitvalue
63  * CborErrorUnknownSimpleType \omitvalue CborErrorUnknownTag \omitvalue
64  * CborErrorInappropriateTagForType \omitvalue CborErrorDuplicateObjectKeys \value
65  * CborErrorInvalidUtf8TextString Illegal UTF-8 encoding found while parsing CBOR
66  * Text String \value CborErrorTooManyItems Too many items were added to CBOR
67  * map or array of pre-determined length \value CborErrorTooFewItems Too few
68  * items were added to CBOR map or array of pre-determined length \value
69  * CborErrorDataTooLarge Data item size exceeds TinyCBOR's implementation
70  * limits \value CborErrorNestingTooDeep Data item nesting exceeds TinyCBOR's
71  * implementation limits \omitvalue CborErrorUnsupportedType \value
72  * CborErrorJsonObjectKeyIsAggregate Conversion to JSON failed because the key in a
73  * map is a CBOR map or array \value CborErrorJsonObjectKeyNotString Conversion to
74  * JSON failed because the key in a map is not a text string \value
75  * CborErrorOutOfMemory During CBOR encoding, the buffer provided is
76  * insufficient for encoding the data item; in other situations, TinyCBOR failed to
77  * allocate memory \value CborErrorInternalError An internal error occurred in
78  * TinyCBOR
79  */
