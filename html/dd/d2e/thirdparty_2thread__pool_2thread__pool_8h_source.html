66 /**
67  * @brief Wait for all queued jobs to finish
68  *
69  * Will wait for all jobs - both queued and currently running to finish.
70  * Once the queue is empty and all work has completed, the calling thread
71  * (probably the main program) will continue.
72  *
73  * Smart polling is used in wait. The polling is initially 0 - meaning that
74  * there is virtually no polling at all. If after 1 seconds the threads
75  * haven't finished, the polling interval starts growing exponentially
76  * until it reaches max_secs seconds. Then it jumps down to a maximum polling
77  * interval assuming that heavy processing is being used in the threadpool.
78  *
79  * @example
80  *
81  * ..
82  * threadpool thpool = thpool_init(4);
83  * ..
84  * // Add a bunch of work
85  * ..
86  * thpool_wait(thpool);
87  * puts("All added work has finished");
88  * ..
89  *
90  * @param threadpool the threadpool to wait for
91  * @return nothing
92  */
93 void thpool_wait(threadpool);
